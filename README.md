## ☕ HARZ Coffee Shop — Web Platform

This is my first serious commercial project, developed for a real coffee shop.

The main goal was to build a modern, clean, and easy-to-manage website for selling coffee, managing course bookings, and handling roasting service requests.

---

## 🚀 Features

The project is built as a full-stack web application with a strong focus on usability, minimalistic design, and real business needs.

### 🛍️ Store

* Coffee catalog with product cards
* Sorting by name and price
* Shopping cart with total calculation
* Order form (name, phone, messenger, comment)
* Orders are sent directly to the administrator

---

### 🎓 Courses

* Dedicated courses section
* Each course includes image, description, and price
* Modal form for submitting course requests
* Separate request handling from store orders

---

### 🔥 Roasting Service

* Service section with description and image
* Custom request form
* Users can submit detailed roasting requests

---

### 📍 Location & Contacts

* Embedded map with location
* Contact information (phone, messenger)
* Quick navigation for users

---

## ⚙️ Admin Panel (Messenger-based)

One of the key features of this project is a fully implemented admin system that works entirely inside a messaging app (no traditional web admin panel).

Through the messenger, the admin can:

* Add, edit, and delete products
* Add and edit courses
* Update roasting service content
* Change **all images and descriptions on the website**
* Receive all orders and requests in real time

This approach allows managing the entire website directly from a phone or computer, making it extremely convenient and fast.

---

## 🧑‍💻 Tech Stack

* **Frontend:** React + Vite
* **Backend:** Node.js (Express)
* **Database:** Firebase
* **Integration:** Messaging bot for admin panel

---

## 💡 Key Highlights

* Real commercial project for a client
* Fully working order flow (store, courses, services)
* Custom admin system without a traditional dashboard
* Mobile-friendly and responsive design
* Clean UI focused on minimalism
